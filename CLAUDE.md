# zf peng portfolio

## 项目概述
React + TypeScript + Tailwind CSS + Framer Motion + Lucide React 构建的个人作品集网站。

## 用户偏好
- 当用户的口述调整有不妥之处时，可以视情况自行调整
- 当有更好的方案时，随时向用户提出建议
- 项目中可以随时调用相关 skill 来辅助设计和实现
- 使用中文与用户交流

## 技术栈
- React 18 + TypeScript
- Vite 6
- Tailwind CSS 3.4
- Framer Motion 12
- Lucide React 0.344
- 字体: Kanit (Google Fonts, 300-900)

## 页面结构
1. HeroSection - 全屏 Hero，固定导航栏，应用图标，卡片堆
2. AboutSection - 关于我，装饰图标，逐字动画
3. ServicesSection - 白色背景服务列表
4. ProjectsSection - 粘性堆叠项目卡片
5. MarqueeSection - Visual Exploration 图片画廊

## 素材路径
- 作品图片: `public/portfolio/`
- 应用图标: `public/icons/`
